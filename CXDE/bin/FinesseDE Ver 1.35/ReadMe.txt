Run Finesse DE 1.0
------------------
1- Open the file CXConnectConfig.properties
2- Modify the following parameters

FinesseA=finesse1.dcloud.cisco.com
FinesseB=finesse1.dcloud.cisco.com
ShowServerDetails=false
3- Run the CXDE application
4- Login with Agent ID and Password and Extension. Please use Agent ID not agent name.
5- Change start and start receiving calls.
6- IPT is supported.
6- Start receiving Instagram comments / feeds.
